### E2E 테스트 설정

공통 / 오류 대응

- 사용자가 필수 입력 누락, 네트워크 오류, 중복 클릭 등 예외 상황을 경험 (의도적으로 필수 값을 누락, 저장 버튼 반복 클릭, 데이터가 없는 날짜 선택)

- 에러 문구, 로딩 상태, 중복 저장 방지, 빈 상태, 권한 / 로그인 정책 점검

월별 소비 리포트 조회

- 사용자가 월별 소비 흐름을 카테고리 / 감정 기준으로 확인

- 사용자가 리포트 화면으로 이동해 월별 총 지출, 카테고리별 지출 비중, 감정별 주요 지출 내역을 확인

회고 설문 및 결과 제공

- 사용자가 만족도와 내일 소비 의도를 선택하고 결과 피드백을 확인한다.

- 사용자는 설문 항목에 응답하고 제출한다

- 시스템은 소비 데이터, 상황/감정 태그, 설문 응답을 조합해 결과 유형과 피드백, 실천 가이드를 제공한다

소비 회고 시작

- 사용자가 하루 소비 요약 확인한 뒤 회고 설문으로 진입한다

- 사용자는 회고 버튼을 클릭한다

- 시스템은 해당 날짜의 지출 데이터를 집계해 총 지출, 최고 지출 카테고리, 주요 감정을 요약해 보여준다.

일별 지출 내역 조회

- 사용자가 특정 날짜의 소비 내역과 총 지출 금액을 확인한다.

- 선택 날짜에 지출 데이터 2건 이상 존재해야 한다.

- 사용자는 가계부 메인에서 날짜를 이동하거나 선택하고, 시스템은 해당 날짜의 전체 지출 금액과 소비 목록을 노출한다.

지출 기록 등록

- 사용자가 오늘 사용한 지출 내역을 감정/상황과 함께 저장한다.

- 사용자는 가계부 화면에서 금액을 입력하고, 날짜, 카테고리, 상황 태그, 감정 태그를 선택한 뒤 저장하고 필요시 메모와 입력 태그도 입력한다.
